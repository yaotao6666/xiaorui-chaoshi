## 初始化数据说明

当前仓库提供单文件初始化模式。

单文件初始化脚本仅保留以下数据：

1. `PRD.md` 中约定的后台账号 `admin / tm666666`
2. `PRD.md` 中约定的商家员工账号 `merchant / merchant123`
3. 账号登录所需的最小后台、门店、配送设置等基础数据

默认初始化脚本不再预置 C 端用户、订单、商品和行为演示数据。

说明：

1. 默认初始化阶段仅保留 `PRD.md` 中约定的后台与商家登录账号。
2. 数据库初始化目录当前只保留 `api-chaoshi/migrations/20240101000000_full_init.sql` 一个有效文件。

## 数据库初始化

### 方式1：直接导入初始化 SQL

```bash
cd api-chaoshi
docker exec -i chaoshi_mysql mysql -uroot -proot123456 < migrations/20240101000000_full_init.sql
```

### 方式2：使用回归脚本

```bash
cd api-chaoshi
./scripts/regression-test.sh init
```

### 方式3：使用 Docker

```bash
cd api-chaoshi

# 先确保容器已启动
docker compose up -d mysql redis

# 导入初始化脚本
docker exec -i chaoshi_mysql mysql -uroot -proot123456 < migrations/20240101000000_full_init.sql
```

## 注意事项

1. **仅用于开发测试**：初始化数据仅供开发和测试环境使用，不要在生产环境使用
2. **密码安全**：生产环境请重置默认密码，使用强密码
3. **数据清理**：如需清空并重新初始化，请使用 `./scripts/regression-test.sh reset`
4. **API地址**：开发环境接口地址通常为 `http://localhost:8080`
